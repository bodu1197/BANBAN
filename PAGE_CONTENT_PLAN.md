## 구현 우선순위

| 순위 | 작업 | 이유 |
|------|------|------|
| **1** | 홈 페이지 콘텐츠 채우기 | 첫인상, 서비스의 얼굴 |
| **2** | 타투 페이지 필터/검색 추가 | 핵심 탐색 기능 |
| **3** | 랭킹 페이지 구현 | DB에 좋아요 데이터 존재, 바로 구현 가능 |
| **4** | 남성/여성뷰티 카테고리 탭 | 세분화된 탐색 경험 |
| **5** | 기획전 페이지 | 운영 콘텐츠 필요 (관리자 기능 선행) |

## 공통 UI 패턴

- **가로 스크롤 리스트**: 기존 `HorizontalScrollList` 컴포넌트 재사용
- **그리드 2열**: 기존 `GridPortfolioCard` 패턴 재사용
- **섹션 헤더**: 기존 `SectionHeader` (제목 + 더보기 링크) 재사용
- **카테고리 탭**: 기존 `CategoryTabs` 컴포넌트 활용
- **데이터 페칭**: 서버 컴포넌트 (ISR, revalidate=60) 기본
